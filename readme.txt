== Description ==
This plugin is an URL Shorter for **WordPress** which filter the shortlink function and return a [bit.ly](https://bitly.com/ "Bitly's website") hyperlink

== Installation ==
1. Upload `wp-shortly` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Activate your [http://dev.bitly.com/](https://bitly.com/ "Bitly's developer website") API
